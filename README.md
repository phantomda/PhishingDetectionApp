# PhishingDetectionApp

## Overview
This project is an AI-powered phishing detection system that classifies emails and URLs as phishing or legitimate using Natural Language Processing (NLP).

## Features
- Real-time phishing detection
- Confidence score for predictions
- Explainable AI (highlights key words influencing decisions)
- Interactive web interface using Streamlit

## Model
- TF-IDF Vectorization
- Logistic Regression classifier

## Dataset
A phishing email dataset containing labeled examples of phishing and legitimate messages.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run

```bash
pip install streamlit
streamlit run app.py
